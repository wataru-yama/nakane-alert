# 仲値アラート Vercelデプロイ手順

このフォルダ（`vercel/`）をそのままVercelにデプロイすると、誰でもブラウザで判定を見られるWebアプリになります。無料プラン（Hobby）で動作します。

## 構成

- `index.html` — フロントエンド（日付選択・判定表示・60秒自動更新）
- `api/judge.py` — サーバレス関数。GMOコイン外国為替FX公開API（認証不要）から当日+過去16日分のレートを取得し、特徴量計算→3ロジック判定をJSONで返す
- `api/l1_model.json` — L1ロジスティック回帰パラメータ
- `requirements.txt` — Python依存（jpholidayのみ）
- `vercel.json` — 東京リージョン(hnd1)・タイムアウト30秒設定

## デプロイ方法A: GitHub連携（推奨・更新が楽）

1. このフォルダをGitHubリポジトリにpush
2. https://vercel.com/new でそのリポジトリをImport（設定はデフォルトのままでOK、Framework Preset: Other）
3. Deploy → 発行されたURL（`https://xxx.vercel.app`）を共有

以後、GitHubにpushするたび自動で再デプロイされます。

## デプロイ方法B: CLI

```bash
npm i -g vercel
cd vercel
vercel --prod
```

## 動作確認

- `https://<あなたのURL>/` — 画面表示
- `https://<あなたのURL>/api/judge?date=2026-07-10` — JSON直接確認（ゴトー日→L1判定が返ればOK）

## 注意事項

- **データソース**: GMOコインの為替公開APIで、APIキー不要・無料・リアルタイムです。バックテストのデータ（GMOクリック証券）とは配信元が異なり価格が1pips弱ずれることがありますが、判定はATR正規化した特徴量を使うため影響は軽微です（過去日照合でCSV版ツールと判定一致を確認済み）。
- **判定タイミング**: 祝日=9:00直前 / 非ゴトー日=9:25直前 / ゴトー日=9:52頃にアクセス。ゴトー日は9:00〜9:53の値動きを使うため、早すぎる時刻だと警告が出ます。
- APIのメンテナンス時間帯や市場休場時はデータ取得エラーになります。
- URLは公開されます。限定共有したい場合はURLを推測されにくい名前にするか、Vercelのアクセス制御を検討してください。
- 本アプリは検証目的の参考情報であり投資助言ではありません。
